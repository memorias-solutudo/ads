# Notas de projeto — decisões e padrões a aplicar

## Termo-chave: "Objetivo"

O campo **Objetivo** é o destaque principal de cada card de vídeo no sistema.
É a primeira tag que aparece no painel e a que responde de imediato: **pra que serve este vídeo?**

Não é a etapa do funil, não é a jornada — é a intenção central do vídeo em uma frase curta.

### Exemplos de objetivos (dos vídeos analisados)

| ID | Objetivo |
|---|---|
| c1 | Conscientizar → diagnóstico (WhatsApp/bot) |
| c2 | Quebrar objeção de confiança (remarketing) |
| c3 | Explicar o que é a Solutudo (awareness) |
| c4 | Agitar dor → consulta pelo bot (WhatsApp) |
| c5 | Cadastro gratuito — espontâneo/UGC |
| c6 | Cadastro gratuito — urgência/volume |
| c7 | Cadastro gratuito + semente de upsell |
| c11 | Agendamento de reunião gratuita (high ticket) |
| c13 | Cadastro gratuito — construção civil |

### Regra de escrita do Objetivo

- Máximo 1 linha
- Começa com um verbo de intenção: conscientizar, converter, aquecer, cadastrar, agendar, vender, reter…
- Termina com o mecanismo de destino quando relevante: (WhatsApp/bot), (página VSL), (agendamento), (checkout), etc.

## Padrão de tag visual

No painel do card (mind map):
1. **Objetivo** — tag de destaque no topo (cor diferenciada)
2. Jornada / Etapa
3. Público
4. Estilo / Formato
5. Plataforma / Campanha
6. KPI
7. Botões de vídeo (▶ Drive / ▶ A Voz / ▶ YouTube)

## Outras decisões de classificação registradas

- A pergunta correta não é "qual etapa do funil?" — é "o CTA deste vídeo convida p/ qual ação, e essa ação leva p/ qual mecanismo de conversão?"
- c2 NÃO vai p/ público frio (4min de prova social sem contexto não funciona).
- c3 precisa de botão de destino definido na plataforma antes de rodar.
- c4 CTA é consultivo ("vai te explicar"), não checkout direto → destino WhatsApp/bot.
- c11 CTA é "reunião gratuita" = agendamento = high ticket, não "apenas confiança".
